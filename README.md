## INSTALLATION ##

### Install node-db-mysql ###

#### Using npm ####

    $ npm install db-mysql

#### Using GIT ####

    $ git clone https://github.com/mariano/node-db-mysql.git
    $ git submodule update --init
    $ cd node-db-mysql
    $ npm install
